## Node

## NPM-modules
    $ npm install
    $ npm install -g nodemon
    
## Client side JS (Jam)
    $ npm install -g jam
More information can be found at: http://jamjs.org/


## CSS/Compass
    $ gem install compass
    $ gem install animation --pre
    $ compass watch sass/screen.scss
More information can be found at http://compass-style.org/
